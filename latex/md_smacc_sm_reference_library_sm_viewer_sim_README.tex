\doxysubsection*{State Machine Diagram}



\doxysubsection*{Description}

A state machine example used for testing and debugging the SMACC viewer. Not recommended for normal SMACC users.~\newline
 \href{https://reelrbtx.github.io/SMACC_Documentation/master/html/namespacesm__viewer__sim.html}{\texttt{ Doxygen Namespace \& Class Reference}}

\doxysubsection*{Operating Instructions}

After you build, remember to source the proper devel folder...


\begin{DoxyCode}{0}
\DoxyCodeLine{source \string~/catkin\_ws/devel/setup.bash}

\end{DoxyCode}


And then run the launch file...


\begin{DoxyCode}{0}
\DoxyCodeLine{roslaunch sm\_viewer\_sim sm\_viewer\_sim.launch}

\end{DoxyCode}
 If you have the SMACC Viewer installed then type...


\begin{DoxyCode}{0}
\DoxyCodeLine{rosrun smacc\_viewer smacc\_viewer\_node.py}

\end{DoxyCode}


If you don\textquotesingle{}t have the SMACC Viewer installed, click \href{http://smacc.ninja/smacc-viewer/}{\texttt{ here}} for instructions.
