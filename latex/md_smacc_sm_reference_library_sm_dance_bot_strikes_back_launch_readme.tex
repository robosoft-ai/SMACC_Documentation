\begin{DoxyCode}{0}

After that you have to build the smacc ros workspace using catkin\+\_\+make.


\begin{DoxyCode}{0}
\DoxyCodeLine{catkin\_make}

\end{DoxyCode}

\begin{DoxyCode}{0}

\end{DoxyCode}
